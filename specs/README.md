# Specs Directory

This directory contains all feature specifications created using Spec-Kit.

## Structure

Each spec is organized in its own directory:

```
specs/
├── 001-feature-name/
│   ├── spec.md           # Feature specification
│   ├── plan.md           # Implementation plan
│   ├── tasks.md          # Task breakdown
│   └── data-model.md     # Data model (if needed)
└── 002-another-feature/
    └── ...
```

## Workflow

1. `/speckit.constitution` - Define project principles
2. `/speckit.specify` - Create feature specification
3. `/speckit.plan` - Design implementation plan
4. `/speckit.tasks` - Break down into tasks
5. `/speckit.implement` - Execute implementation
